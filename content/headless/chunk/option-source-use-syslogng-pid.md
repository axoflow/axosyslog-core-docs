## use-syslogng-pid()

|          |                 |
| -------- | --------------- |
| Type:    | `yes`, `no` |
| Default: | `no` |

*Description:* When enabled, {{< product >}} overrides the PID from the original message with the PID of the `syslog-ng` process.
