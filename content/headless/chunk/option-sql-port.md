## port()

|          |                                                                                      |
| -------- | ------------------------------------------------------------------------------------ |
| Type:    | number                                                                               |
| Default: | 1433 TCP for MSSQL, 3306 TCP for MySQL, 1521 for Oracle, and 5432 TCP for PostgreSQL |

*Description:* The port number to connect to.
