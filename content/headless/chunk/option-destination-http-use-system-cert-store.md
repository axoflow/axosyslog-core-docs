## use-system-cert-store()

|          |          |
| -------- | -------- |
| Type:    | yes | no |
| Default: | no       |

*Description:* Use the certificate store of the system for verifying HTTPS certificates. For details, see the [curl documentation](https://curl.haxx.se/sslcerts.html).
