## so-broadcast()

|          |           |
| -------- | --------- |
| Type:    | yes or no |
| Default: | no        |

*Description:* This option controls the `SO_BROADCAST` socket option required to make `syslog-ng` send messages to a broadcast address. For details, see the `socket(7)` manual page.
