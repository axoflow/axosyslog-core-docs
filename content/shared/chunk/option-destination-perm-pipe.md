## perm()

|          |                         |
| -------- | ----------------------- |
| Type:    | number (octal notation) |
| Default: | 0600                    |

*Description:* The permission mask of the pipe. For octal numbers prefix the number with '0', for example: use 0755 for rwxr-xr-x.
